# Omega

A C++ sequencing engine and MIDI library. The foundation for building serious music software.

Omega is a library, not an application. It handles timing, event scheduling, multi-track and pattern sequencing, and live performance control — everything needed to build a sequencer, with no opinion about what the interface looks like or what protocols it speaks.

See [PROPOSAL.md](PROPOSAL.md) for the full design document.
